# FIAP-5-Datathon
Trabalho de pós graduação FIAP  

Análise Educacional e Predição de Risco de Defasagem  

Este projeto foi desenvolvido com o objetivo de criar uma solução baseada em Data Analytics e Machine Learning para apoiar a ONG Passos Mágicos na identificação de alunos em risco de defasagem educacional, a partir de indicadores acadêmicos, comportamentais e psicossociais.  

# Objetivo  

Desenvolver uma análise estratégica dos dados educacionais, identificar os principais fatores que impactam o desempenho dos alunos e construir um modelo preditivo capaz de antecipar riscos, além de disponibilizar essa solução por meio de uma aplicação interativa para apoio à tomada de decisão.  

# Base de Dados  

A base de dados utilizada contém informações relacionadas ao desenvolvimento dos alunos ao longo do tempo, incluindo indicadores como desempenho acadêmico (IDA), engajamento (IEG), aspectos psicossociais (IPS), avaliação psicopedagógica (IPP), autoavaliação (IAA) e índice geral de desempenho (INDE).  

# Pipeline de Análise e Machine Learning  

O projeto seguiu uma abordagem completa de análise de dados e modelagem preditiva, incluindo:  

Análise exploratória dos dados  
Tratamento e padronização das variáveis  
Análise de correlação entre indicadores  
Geração de insights estratégicos  
Feature engineering    
Treinamento de modelo supervisionado  
Avaliação do modelo preditivo  

O modelo desenvolvido permite estimar a probabilidade de um aluno entrar em situação de risco de defasagem, possibilitando ações preventivas por parte da instituição.  

# Aplicação Preditiva  

Foi desenvolvida uma aplicação utilizando Streamlit, permitindo que usuários insiram os indicadores do aluno e recebam a previsão de risco em tempo real de forma simples e intuitiva.  

A aplicação atua como uma ferramenta de apoio à decisão, permitindo priorizar alunos com maior necessidade de intervenção e direcionar estratégias educacionais de forma mais eficiente.  

# Dashboard Analítico  

Além da aplicação preditiva, foram desenvolvidas análises visuais e gráficos que permitem compreender o comportamento dos indicadores ao longo do tempo, identificar padrões de evolução e entender os principais fatores que impactam o desempenho educacional.  

Os principais insights incluem:  

Evolução consistente do desempenho médio dos alunos  
Existência de um grupo relevante em situação de risco  
Forte relação entre engajamento e desempenho  
Influência significativa de fatores psicossociais  
Necessidade de abordagem multidimensional no acompanhamento dos alunos
